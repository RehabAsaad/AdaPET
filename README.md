# AdaPET
🐾 Pet Adoption & Veterinary Management System
A web-based platform built using ASP.NET MVC and Entity Framework Core to manage pet adoption and veterinary clinic operations in a structured system.
The application allows users to register and manage pets, track their adoption status, and maintain ownership records. Each pet is linked to an owner, who can update its adoption status when necessary.

The system also manages veterinarians and clinics, where doctors can register, define their specialization, and be associated with multiple clinics. Clinics can also host multiple doctors, forming a many-to-many relationship.

This project follows a clean MVC architecture with a well-structured relational database design, focusing on scalability and future expansion such as appointment booking and adoption workflows.

🚀 Features
User registration and management
Pet creation and adoption status tracking
Doctor profiles with specialization
Clinic management system
Many-to-many relationship between doctors and clinics
Clean MVC architecture with Entity Framework Core
🛠️ Tech Stack
ASP.NET MVC
Entity Framework Core
SQL Server
C#
